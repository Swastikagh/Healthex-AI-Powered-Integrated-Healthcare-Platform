Healthex – AI-Powered Integrated Healthcare Platform

One Platform. Complete Care.

Healthex is an intelligent digital healthcare platform that integrates real-time health monitoring, AI-based insights, doctor consultation, and pharmacy services into a single unified system. It aims to eliminate fragmented healthcare experiences and improve accessibility, efficiency, and patient outcomes.

🚀 Features
🩺 Health Monitoring
Track vital parameters:
1. Heart Rate
2. Blood Pressure
3. Temperature
4. SpO₂
Interactive dashboards with data visualization
🤖 AI Health Insights
1. Analyzes health data in real-time
2. Detects anomalies using medical thresholds
3. Generates health scores
4. Provides personalized recommendations
👨‍⚕️ Doctor Consultation
1. Browse doctors by specialization
2. Book appointments seamlessly
3. Digital consultation workflow
💊 Online Pharmacy
1. Upload prescriptions securely
2. Order medicines online
3. Track orders and delivery status
🏗️ System Architecture

Healthex follows a 3-layer architecture:

1. Frontend
Built with React + TypeScript
Provides interactive UI and dashboards
2. Backend
Powered by Supabase
Handles authentication, APIs, and real-time updates
3. Database
Uses PostgreSQL
Stores user data, health records, appointments, and prescriptions
🔄 Data Flow
User → Frontend → Backend (Supabase) → Database (PostgreSQL)
     ←──────────── Response & Updates ────────────
🛠️ Tech Stack
Frontend: React, TypeScript
Backend: Supabase
Database: PostgreSQL
Visualization: Charts & Graphs
